# CreativeTech Digital Agency Website

CreativeTech is a modern, responsive multi-page website built for a forward-thinking digital agency. This project showcases the agency's services, team, mission, and company journey from 2015 to the present.

## ✨ Features
* **Fully Responsive:** Optimized for desktops, tablets, and mobile devices.
* **Multi-Page Architecture:** Includes Home, Services, About, and Contact pages.
* **Interactive UI:** Modern navigation bar with active state tracking and hover effects.
* **Service Comparison:** A detailed pricing/service plan table for business transparency.
* **Dynamic Sections:** Features a company timeline, team profiles, and achievement counters.
* **Contact Integration:** Includes a functional UI for contact forms and Google Maps integration.
